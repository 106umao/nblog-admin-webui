# nblog-admin-webui

>博客系统后台管理模块webui


**演示案例:** http://panjiachen.github.io/vue-admin-template


**项目基于nodejs vue3.0 vue-cli**

## 构建安装


```bash
# clone the project
git clone https://github.com/106umao/nblog-admin-webui.git

# enter the project directory
cd blog-admin

# install dependency
npm install

# develop
npm run dev
```

运行后程序会自动在浏览器中打开 http://localhost:9528
